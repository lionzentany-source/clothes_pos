Clothes POS - Windows Release
=============================

Arabic (العربية)
----------------
طريقة التشغيل:
1. فك الضغط عن الملف.
2. شغّل الملف: clothes_pos.exe
3. في أول تشغيل سيتم إنشاء مجلدات وملفات إعداد محلياً (إن لزم).

متطلبات:
- نظام Windows 10 أو أحدث (64-bit).
- وجود اتصال طابعة (اختياري للإيصالات الحرارية / PDF يمكن حفظه فقط).

ملاحظات:
- البيانات مخزنة محلياً (SQLite) داخل مجلد التطبيق أو ضمن المسار المؤقت الخاص بالمستخدم.
- احرص على أخذ نسخة احتياطية من ملفات *.db قبل الترقية.

حل مشاكل شائعة:
- في حال منع التشغيل من SmartScreen: اختر المزيد ثم تشغيل على أي حال (يمكن لاحقاً توقيع الملف رقمياً لتفادي الرسالة).
- في حال فشل الطباعة الحرارية: تأكد من تعريف الطابعة أو استخدام وضع PDF.

English
-------
Run Steps:
1. Extract the archive.
2. Launch clothes_pos.exe.
3. On first run local data/db folders may be created.

Requirements:
- Windows 10+ (64-bit).
- Printer (optional). Thermal receipts require a compatible ESC/POS printer.

Notes:
- Data is stored locally (SQLite). Backup *.db files before upgrading.
- Unsigned build; SmartScreen may warn. This is expected for internal/private distribution.

Troubleshooting:
- If blocked by SmartScreen: More Info -> Run Anyway.
- Printing issues: install correct printer driver or test PDF export.

Support / Internal Use Only.
